# PRD Front Matter Example

Follow `references/front_matter_prd.md` (`shared/references/front_matter_prd.md`) schema:

```yaml
---
id: "prd-{feature-name}"
title: "{Feature Title}"
type: "prd"
status: "draft"
created: "{YYYY-MM-DD}"
updated: "{YYYY-MM-DD}"
depends-on: []  # Parent PRD if hierarchical
priority: "medium"  # or extract from requirements
risk: "medium"  # or extract from requirements
tags: ["{tag1}", "{tag2}"]
category: "{category}"
---
```
